# Tetris

| Game Action | Keyboard | Numpad |
| -------------| ------------- |  ------------- |
| Move Left   | LEFT Arrow     | Numpad 4    |
| Move Right     | RIGHT Arrow       | Numpad 6     |
| Drop     | Space Bar       | Numpad 8      |
| Rotate Right     | Up Arrow, X       | Numpad 1, 5, 9      |
| Rotate Left     | Contro, Z       | Numpad 3, 7      |
| Pause     | P       |     |
